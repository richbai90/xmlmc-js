xmlmc

xmlmc-js

A javascript wrapper for the Supportworks XMLMC API. Currently still in beta stage. To use run

npm install xmlmc
const Xmlmc = require('xmlmc');

Usage

The package is built for a node server. To use in a web browser, you will need to use a bundler like Webpack. Below is an example of using the package in a node environment.

const xmlmc = new Xmlmc('https://devserver.local');
xmlmc.session.analystLogon('admin', 'password').then(() => {
    ...
}).catch(promiseCatcher)

function promiseCatcher(e) {
    console.log(e);
}

The default export is the XmlMethodCall class
